Trivy Security Demo

Container & Source scanning — demo app for vulnerability reporting

About this demo

This small static app demonstrates a working web page served by Nginx inside a Docker container. We use Trivy to scan the container image and the application files.

Actions

Open a terminal in this project folder and run: trivy fs .
